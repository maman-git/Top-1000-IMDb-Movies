# Top-1000-IMDb-Movies

Top 1000 IMDb Movies – Power BI Analytics Dashboard

This repository showcases a full-stack Business Intelligence workflow built on a dataset of the Top 1000 IMDb movies. The objective was to build an end-to-end analytical product—from raw data ingestion to executive-grade insights—using industry-standard BI methodologies.


# Project Overview

The dashboard delivers a breakdown of IMDb’s top-ranked movies, enabling users to analyze:
- Rating distribution
- Genre performance
- Revenue trends
- Release year patterns
- Director/actor impact metrics
- Comparative KPIs across decades

The solution is engineered to support both technical analysts and business stakeholders with a clean UI, intuitive navigation, and data-driven storytelling.


# Core Skills Demonstrated
1. Data Modeling and Relationship Management:

Built a star-schema-aligned data model with optimized one-to-many and many-to-many relationships.
Enforced data integrity and performance through normalization and structured relationship design.

2. Advanced DAX:

Used DAX to construct dynamic analytical logic, including:
- Time-intelligence calculations
- KPI metrics (Avg Rating, Trend Variance, Rank Measures)
- Conditional measures for slicing and drill-downs
- Performance-optimized calculated columns and measures

3. Power Query ETL:

Executed a robust data-shaping process:
- Column transformations
- Data type enforcement
- Outlier handling
- Standardization of categorical fields

4. Data Visualization Strategy:

Deployed modern BI design principles to ensure:
- Detailed drill-downs for analytical deep dives
- Clean UX with intuitive navigation
- Storytelling visuals that highlight patterns, anomalies, and trends
